## üí° PRO TIPS FOR LOOM

**Audio Quality:**
- Speak clearly (not too fast, not too slow)
- Use natural pauses between sentences
- If background noise appears, mention it after stopping and Loom might offer to filter it

**Recording Tips:**
- Don't worry about perfection - first takes are usually fine
- If you make a mistake, pause 2 seconds and repeat
- You can retake any section - just stop and record again
- Loom will keep your latest recording

**Video Settings:**
- Loom automatically records in Full HD (1920x1080)
- Automatic captions are added (optional for viewers)
- You can edit video later if needed (trim, add text, etc.)

**Sharing:**
- Link never expires
- You can track who watched and for how long
- Recipient doesn't need Loom account
- Password protection available (if sensitive)

## ‚ùì TROUBLESHOOTING

**Problem: "Can't hear my microphone"**
- Check browser permissions (allow microphone access)
- Check system sound settings
- Try different microphone
- Restart browser

**Problem: "Website not loading"**
- Check npm start is still running
- Go to: http://localhost:3000
- If still error, check terminal for messages

**Problem: "Recording paused or stopped"**
- Check internet connection
- Check if Loom crashed (refresh page)
- You can restart - Loom will keep previous recordings

**Problem: "Audio out of sync"**
- Not a problem with Loom (it's simultaneous)
- If upload is slow, wait for processing
- Refresh page and re-check

**Problem: "Can't copy link"**
- Refresh page after video finishes processing
- Try different browser
- Check your Loom dashboard (may need to navigate there)
