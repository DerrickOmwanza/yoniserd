================================================================================
                       TROUBLESHOOTING GUIDE
================================================================================

PROBLEM: Blank white page after deployment
SOLUTION: Ensure src/App.js has basename="/ynis-rd-website"

PROBLEM: Assets not loading (broken images, CSS)
SOLUTION: Check homepage in package.json is correct

PROBLEM: 404 Page Not Found error
SOLUTION: 
  - Wait 2-3 minutes for GitHub Pages to process
  - Clear browser cache (Ctrl+Shift+Delete)
  - Refresh page (Ctrl+F5)

PROBLEM: Links not working on site
SOLUTION:
  - Verify Router has basename="/ynis-rd-website"
  - Run npm run deploy again

PROBLEM: Mobile menu not showing links
SOLUTION: Already fixed in previous commits - should work now

PROBLEM: Deployment script fails
SOLUTION:
  - Check you have GitHub credentials set up
  - Run: git config --global user.email "your@email.com"
  - Run: git config --global user.name "Your Name"
  - Try again: npm run deploy
