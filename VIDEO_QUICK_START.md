## OPTION 1: FASTEST ‚ö° (Loom - 15 minutes total)

### What is Loom?
Free screen recording software that records both screen and voice in one go. No editing needed.

### Steps:

1. **Go to:** https://www.loom.com
2. **Sign up** with email (takes 2 min)
3. **Install Loom** extension (1 min)
4. **Open your website** at localhost:3000 (make sure `npm start` is running)
5. **Click Loom icon** ‚Üí "Start recording"
   - Choose "Entire screen"
   - Allow microphone access
   - Click "Start recording"
6. **Narrate while browsing the website:**
   - Read from the WEBSITE_NARRATION_SCRIPT.md
   - Spend 40-60 seconds on each page
   - Speak clearly and naturally
7. **When done, click "Stop"**
8. Loom auto-processes (2 min wait)
9. **Share the link** with stakeholders
   - Click "Share" ‚Üí copy link
   - Send via email
   - No file size worries!

**Pros:**
- ‚úÖ Easiest option
- ‚úÖ No software to install (browser-based)
- ‚úÖ Auto cloud storage
- ‚úÖ Easy sharing (just a link)
- ‚úÖ Auto captions available

**Cons:**
- ‚ùå Requires internet connection
- ‚ùå Limited to free tier (5-10 min recordings)

## OPTION 2: EASIEST ON WINDOWS ü™ü (Built-in Tools - 30 minutes total)

### No software needed! Use Windows features you already have.

### Steps:

1. **Make sure website is running:**
   ```
   npm start
   ```
   Opens at: http://localhost:3000

2. **Record screen:**
   - Press **Windows Key + G** (Xbox Game Bar)
   - Click "Start Recording" or press **Windows Key + Alt + R**
   - You'll see recording indicator in top-left

3. **Navigate website slowly** (while recording):
   - Home ‚Üí About (scroll all sections) ‚Üí Programs ‚Üí Our Work ‚Üí Impact ‚Üí News ‚Üí Contact
   - Take 40-60 seconds per page
   - Scroll smoothly

4. **Stop recording:**
   - Press **Windows Key + Alt + R** again
   - Video auto-saves to: Videos > Captures

5. **Record narration separately:**
   - Open "Voice Recorder" (search Windows)
   - Click record
   - Read WEBSITE_NARRATION_SCRIPT.md slowly and clearly
   - Click stop
   - Saves automatically

6. **Combine them (Choose A or B):**

   **A) Using DaVinci Resolve (Free, Professional):**
   - Download: https://www.blackmagicdesign.com/products/davinciresolve
   - Create new project
   - Drag video to timeline
   - Drag audio below it
   - Align them
   - File ‚Üí Export ‚Üí H.264 MP4
   - Save as "YNIS_RD_Video.mp4"

   **B) Using free online tool:**
   - Go to: https://www.kapwing.com/tools/video-editor
   - Upload video + audio
   - Drag them to timeline
   - Export as MP4

**Pros:**
- ‚úÖ No extra software initially
- ‚úÖ Free
- ‚úÖ Built-in Windows quality

**Cons:**
- ‚ùå Need to record audio separately
- ‚ùå Need to combine video + audio (requires one free tool)

## OPTION 3: MOST PROFESSIONAL üé¨ (OBS Studio - 25 minutes total)

### OBS = Professional livestream/recording software (Free)

### Steps:

1. **Download OBS Studio:**
   - https://obsproject.com/
   - Install on your computer (takes 3 min)

2. **Launch OBS**

3. **Add your screen:**
   - Click "+" under Sources
   - Select "Display Capture"
   - Choose your main monitor
   - Click OK

4. **Add your microphone:**
   - Click "+" under Sources
   - Select "Audio Input Capture"
   - Choose your microphone
   - Click OK

5. **Click "Start Recording"**

6. **Talk and navigate (same as above):**
   - Use WEBSITE_NARRATION_SCRIPT.md
   - Navigate website pages
   - Speak clearly while recording
   - Spend 40-60 sec per page

7. **Click "Stop Recording"**
   - Video automatically saved (usually Documents folder)
   - File is named something like "recording.mp4"

8. **Share directly** - File is ready!

**Pros:**
- ‚úÖ Professional quality
- ‚úÖ Free
- ‚úÖ Records audio + video simultaneously
- ‚úÖ No editing needed

**Cons:**
- ‚ùå Need to install software
- ‚ùå Takes a few minutes to learn UI

## NARRATION QUICK TIPS

**While recording:**
- Speak clearly and at normal pace (not too fast)
- Pause between sentences (sounds more professional)
- Show enthusiasm - this is great work!
- Emphasize numbers: "Ten THOUSAND youth"
- If you mess up, pause 2 seconds and repeat the sentence
- Smile while talking - it affects your voice tone!
