  [ ] Restart browser
  [ ] Restart computer

ISSUE: "Website not loading at localhost:3000"
ACTION:
  [ ] Confirm npm start is still running
  [ ] Try: npm start again in terminal
  [ ] Check terminal for error messages
  [ ] Try different port if suggested
