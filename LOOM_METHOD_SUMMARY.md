## üéôÔ∏è NARRATION TIPS

**While recording, remember:**
- Speak at natural pace (120-150 words/min)
- Use natural pauses between sentences
- Emphasize key numbers: "Seventy-FIVE percent"
- Smile while talking (improves voice tone)
- Don't rush - take your time
- If you mess up, pause and repeat
- Scroll slowly so viewers can read
- Show genuine enthusiasm

## üÜò IF SOMETHING GOES WRONG

**Problem: Microphone not working**
- Check browser permissions
- Check system sound settings
- Restart browser
- Try different microphone

**Problem: Website not loading**
- Confirm: npm start running
- Check: localhost:3000 in browser
- Restart npm if needed

**Problem: Recording stopped**
- Check internet connection
- Close other heavy apps
- Try recording again
- Loom saves your recordings

**Problem: Audio too quiet/loud**
- Adjust microphone input level
- Move mic 6-12 inches from mouth
- Check system volume
- Re-record if necessary

**More help:** See LOOM_QUICK_GUIDE.md troubleshooting section
